Code. Compile. Run.

Code App is a desktop-class IDE for iOS / iPadOS with built-in cloud compilers support for 40+ languages.

by thebaselab

Cloud Compiling for 40+ Languages

Code App comes with free server-side compiling service with multi-file support, letting you inspect and debug your local code on the go.

Monaco Editor Integration

With Monaco Editor integration, Code App shares most editing features found in the desktop VS Code, this includes find and replace, command palatte, word-based Intellisense, multi-cursor and more.

Cloning a repository from GitHub

Simply enter the link of a GitHub repository, Code App automatically creates a copy of the repository on your device.

Built-in web server

Preview your website with a local web server with asynchronous requests support. Similarly to Python's SimpleHTTPServer, you can access the webpage on any browser with the app opened side by side.
